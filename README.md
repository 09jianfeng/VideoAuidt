### 分析
与言论相比
1. 制作一个视频的成本更大,很少有人会因为非利益驱动而制作大量非法视频
2. 视频的传播速度快,覆盖面积大       
视频处理主要以**删除,封禁为主;锁定为辅**,尽快减少平台上的恶意视频

低俗视频/色情视频/反动视频



### 核心思路
视频 -> 图片        
图片 -> 指纹(用于快速检索及审核去重)       
图片 - > 恶意程度数值       
色情程度 -> 数值
色情程度 -> 肌肤裸露比例
色情程度 -> 人物造型　


### 核心问题
1. 如何检测出恶意视频
2. 如何防止某特定主题恶意视频集中上传/过审
3. 如何生成一个表明视频特征的指纹
